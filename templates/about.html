Simple Django Blog DEMO

Django web app for users to share their thoughts online.

Features:
*Form class
*Restful APIs

Demo details:
This demo is made from 3 apps:

main_app ()
1. The web application have a dashboard (home), where latest posts will show up. Posts should be ordered by date.
2. In order to post a new thought, you have to be a registered user.
3. User needs to log in to see their profile name on navigation bar.
4. User can post new thoughts from their profile.
5. The posted thoughts will be stored in a MySql database.
6. GET request should get 10 latest posted thoughts in JSON format
7. POST request should store a new post in the database.
(If user is authenticated is allow to submit a post if not he can only see the posts) API Link

user ()

1.Register new users.
2.Login/logout

sendmail ()

1.Contact form

Tis demo has tests for all views from main_app and sendmail
